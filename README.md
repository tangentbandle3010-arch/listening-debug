英語リスニングを「音声データのデバッグ」と捉え、日々の進捗をコミットするリポジトリです。
聞き取れない音（バグ）を特定し、脳内レジストリに正しい音をマッピングすることを目的としています。

## 🔍 Known Bugs (Common Listening Issues)
- [ ] **Liaison (Linking)**: `fears of` → "フィアズォヴ" と聞こえるバグ。
- [ ] **Proper Nouns**: 人名・地名のパースエラー（未知のシンボル）。
- [ ] **Reduction**: `to` や `and` が極端に短くなる現象への対応。

## 🚀 How to Contribute to Myself
1. **Fetch**: 音声を3〜5回聴く（0.8x speed 推奨）。
2. **Code**: エディタに書き出す。
3. **Debug**: スクリプトと Diff を取り、赤色（ミス）を修正する。
4. **Commit**: 修正後の音読を行い、脳内メモリへプッシュする。
